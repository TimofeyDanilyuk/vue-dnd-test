# Client — Vue 3

Фронтенд часть Scheme App. Интерфейс для работы со схемами: авторизация, палитра элементов, рабочее полотно.

## Стек

- Vue 3 (Composition API)
- Vite
- vue-draggable-plus — drag-and-drop на базе Sortable.js
- html-to-image — экспорт полотна в PNG
- axios — запросы к API
- vue-router — навигация

## Страницы

- `/login` — авторизация и регистрация
- `/` — основной редактор схем (требует авторизации)

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение запустится на `http://localhost:5173`. Бэкенд должен быть запущен на порту `5204`.

## Как устроен редактор

**Палитра** — список загруженных пользователем изображений. Подгружается с сервера при старте. Новый элемент добавляется через кнопку "Добавить деталь", изображение отправляется на бэкенд и сохраняется в БД.

**Полотно** — рабочая область с сеткой шагом 80px. Элементы перетаскиваются из палитры и автоматически привязываются к сетке. При перекрытии элемент ищет свободное место через BFS-обход.

**Экспорт** — кнопка "Сохранить PNG" снимает скриншот полотна через `html-to-image`.

## Авторизация

JWT токен сохраняется в `localStorage` и автоматически добавляется в каждый запрос через axios interceptor. При отсутствии токена роутер перенаправляет на `/login`.
